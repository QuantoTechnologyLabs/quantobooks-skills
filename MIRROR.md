# About this repo

This is a **read-only mirror** of the QuantoBooks Claude skills bundle.

Source of truth: [glaceon-monorepo](https://github.com/quantotechnologylabs/glaceon-monorepo) at `apps/mcp-server/skills/`.

Pull requests opened against this mirror will be closed without review. Please open them against the source repo.

## How to install

Shell (macOS / Linux):
```bash
curl -fsSL https://get.quantobooks.com/skills | bash
# or, without the redirect domain:
curl -fsSL https://raw.githubusercontent.com/quantotechnologylabs/quantobooks-skills/main/install.sh | bash
```

Node CLI (any OS with Node 18+):
```bash
npx @quantobooks/skills install
```

See `INSTALL.md` for all options.
